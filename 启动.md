# 启动.md

本文件用于在 `E:\excel_agent` 启动前后端服务，后端端口为 `5201`。

## 0. 环境准备

- Windows
- Node.js 18+
- Python 3.10+（可创建 venv）

## 1. 一键启动（推荐）

在根目录双击运行：

- `E:\excel_agent\start.bat`

它会自动：
- 创建 `backend\venv`
- 安装后端依赖 `backend\requirements.txt`
- 安装前端依赖 `frontend\node_modules`
- 启动后端 `http://localhost:5201`
- 启动前端 `http://localhost:3000`

## 2. 手动启动（排查用）

### 2.1 启动后端（端口 5201）

```powershell
cd E:\excel_agent\backend
.\venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --host 127.0.0.1 --port 5201
```

### 2.2 启动前端

```powershell
cd E:\excel_agent\frontend
npm install
npm run dev
```

前端已通过 `frontend\vite.config.js` 将 `/api` 代理到 `http://127.0.0.1:5201`。

## 3. 停止服务

- 前端终端：`Ctrl + C`
- 后端终端：`Ctrl + C`

## 4. 常见问题

- 前端请求失败：确认 `http://localhost:5201/health` 可访问。
- 依赖安装失败：确认 Node/Python 版本符合要求。
